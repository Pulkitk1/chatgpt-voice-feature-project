# chatgpt-voice-feature-project
Led end-to-end user research and problem framing for ChatGPT’s mobile voice feature, identifying a clear gap between high feature awareness and low habitual usage, primarily driven by trust, privacy, and usability friction points.

Authored the PRD with clearly defined goals, target segment (working professionals aged 21–34), and a measurable KPI framework covering adoption, accuracy, latency, reliability, and feature discoverability.
